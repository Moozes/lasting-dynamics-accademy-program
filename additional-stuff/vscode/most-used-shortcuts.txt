General Editor Shortcuts
    Ctrl + P: Quick Open, Go to Fileâ€¦
    Ctrl + Shift + P: Show Command Palette
    Ctrl + ` (Backtick): Toggle integrated terminal
    Ctrl + B: Toggle sidebar visibility
    Ctrl + Shift + N: Open a new window/instance
    Ctrl + W: Close the current window/editor
    
Navigation
    Ctrl + G: Go to Line...
    Ctrl + T: Show all Symbols
    Ctrl + Shift + O: Go to Symbol...
    Ctrl + Shift + M: Focus on Problems panel
    Alt + Left / Alt + Right: Go back / forward

Code Editing
    Ctrl + X: Cut line (empty selection)
    Ctrl + C: Copy line (empty selection)
    Ctrl + V: Paste
    Ctrl + /: Toggle line comment
    Shift + Alt + A: Toggle block comment
    Alt + Up/Down: Move line up/down
    Shift + Alt + Up/Down: Copy line up/down
    Ctrl + Shift + K: Delete line
    Ctrl + Enter: Insert line below
    Ctrl + Shift + Enter: Insert line above

Code Selection & Manipulation
    Ctrl + D: Select the next match of the current selection
    Ctrl + L: Select the current line
    Ctrl + Shift + L: Select all occurrences of the current selection
    Ctrl + F2: Select all occurrences of the current word
    Shift + Alt + Right: Expand selection
    Shift + Alt + Left: Shrink selection

Multi-Cursor and Refactoring
    Alt + Click: Insert cursor
    Ctrl + Alt + Up/Down: Insert cursor above/below
    Ctrl + U: Undo last cursor operation
    Shift + Alt + I: Insert cursor at end of each line selected
    Ctrl + Shift + Alt + (Arrow Key): Column selection mode

Search and Replace
    Ctrl + F: Find
    Ctrl + H: Replace
    F3 / Shift + F3: Find next/previous
    Ctrl + Shift + F: Open global search

Debugging
    F5: Start/Continue
    Shift + F5: Stop
    F10: Step over
    F11: Step into
    Shift + F11: Step out
    Ctrl + K Ctrl + I: Show hover

Version Control
    Ctrl + Shift + G: Open Source Control
    Ctrl + Enter (in Source Control): Commit
    Ctrl + Shift + K: Discard changes
    
File Management
    Ctrl + N: New File
    Ctrl + S: Save
    Ctrl + Shift + S: Save As
    Ctrl + K S: Save All
